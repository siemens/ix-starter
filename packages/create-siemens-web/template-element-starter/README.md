# ElementStarter

Element starter project

## Usage

To locally set up this project, use the following steps:

```sh
# Install all dependencies
npm install

# Start the development server
npm start
```

Navigate to `http://localhost:4200/` with your browser. The app will
automatically recompile and reload if you change any of the source files.

### Running Unit Tests

To execute unit tests with the [Vitest](https://vitest.dev/) test runner, use the following command:

```sh
ng test
```

### Running End-to-End Tests

Run `npm run e2e` to execute the end-to-end tests via [Playwright](https://playwright.dev/).

### Running Code Linting

- Run `npm run lint` for angular linting.
- Run `npm run lint:sass` for sass linting.
